# Zomato SQL Data Analysis Project

## 📌 About the Project

This project is based on a food delivery platform like Zomato.  
I created a relational database and solved different business problems using SQL.

The goal of this project is to practice and demonstrate SQL skills from basic to advanced level.

---

## 🗂 Database Tables

The database contains the following tables:

- users
- restaurants
- food
- menu
- orders
- order_details
- delivery_partner

These tables are connected using primary keys and foreign keys.

---

## 📊 What I Analyzed

### Basic Queries
- Customers who never ordered
- Average price of each dish
- Peak ordering day
- Orders in specific months

### Intermediate Queries
- Restaurants with high monthly sales
- Repeat customers per restaurant
- Most loyal customers
- Second highest revenue restaurant
- Customers spending above average
- Revenue contribution of each restaurant

### Advanced Queries
- Month-over-Month revenue growth
- Restaurant-wise MoM growth
- Customer retention rate
- Customer favorite food (using window functions)
- Most frequently ordered food pairs
- Top 3 restaurants per month

---

## 🧠 SQL Concepts Used

- Joins (INNER JOIN, LEFT JOIN)
- GROUP BY and HAVING
- Subqueries
- CTE (Common Table Expressions)
- Window Functions (RANK, DENSE_RANK, LAG)
- Date functions
- UNION / UNION ALL

---

- SQL Workbench
